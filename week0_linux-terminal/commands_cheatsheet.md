# Linux Terminal Commands – Week 0

## Navigation
- `pwd` – Show current directory
- `ls` – List files
- `ls -la` – List all files with details
- `cd folder_name` – Move into folder
- `cd ..` – Move up one directory
- `cd ~` – Go to home directory

## Files & Folders
- `mkdir folder_name`
- `touch file.txt`
- `cp file1 file2`
- `mv old_name new_name`
- `rm file.txt`
- `rm -r folder_name`

## Editing
- `nano file.txt`
- `CTRL + O` save
- `CTRL + X` exit

## Permissions
- `ls -l`
- `chmod +x script.sh`

## Running Scripts
- `python3 script.py`
- `./script.sh`
